업무 프로그램 다운로드 포털

Codex에서 만든 업무용 프로그램을 한 곳에서 내려받는 정적 포털입니다. 사무실에서는 이 페이지에 접속한 뒤 필요한 프로그램의 압축파일이나 사용방법 문서를 다운로드하면 됩니다.

기본 사용 흐름

  1. 프로그램 카드를 열어 설명을 확인합니다.
  2. 먼저 사용방법 PDF를 열어 입력 파일과 절차를 확인합니다.
  3. 가능하면 `zip` 파일을 내려받아 압축을 해제합니다.
  4. 압축 해제 후 실행 파일을 실행합니다.

앞으로 추가하는 방법

  • `files/프로그램슬러그/` 폴더를 만들고 `zip`, `exe`, `PDF`를 넣습니다.
  • `apps.json`에 프로그램 메타데이터와 다운로드 경로를 추가합니다.
  • GitHub에 push하면 포털 페이지가 그대로 갱신됩니다.
