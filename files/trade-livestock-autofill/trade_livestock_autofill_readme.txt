거래가축 자동완성 프로그램 사용 안내

1. 권장 다운로드는 zip 파일입니다.
2. 압축을 푼 뒤 거래가축_자동완성.exe를 실행합니다.
3. 프로그램이 다운로드 폴더와 현재 폴더에서 가장 최신 채혈대상 조회*.xlsx 파일을 자동으로 찾습니다.
4. 그날 농업e지에서 설정한 비밀번호를 입력합니다.
5. Enter 또는 자동완성 실행 버튼을 누릅니다.

필수 조건
- Microsoft Excel이 설치되어 있어야 합니다.

프로그램이 하는 일
- 채혈대상 조회 파일을 읽어 브루셀라 의뢰 농가명단 엑셀을 자동으로 만듭니다.
- 날짜, 읍면, 성명, 브루셀라 검사두수, 결핵 검사두수, 채혈자를 자동으로 채웁니다.
- 검사 두수가 0이면 해당없음으로 입력합니다.
- 원본 양식의 글꼴, 글자크기, 셀 서식, 인쇄 영역을 유지합니다.

주의
- 비밀번호는 저장하지 않습니다. 담당자가 바뀌면 그날 비밀번호만 다시 입력하면 됩니다.
- 자동으로 선택된 파일이 다르면 다른 파일 버튼으로 직접 바꿀 수 있습니다.
- 저장 폴더를 바꾸고 싶으면 폴더 변경 버튼을 누르면 됩니다.
